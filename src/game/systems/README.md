# Systems

Put reusable gameplay systems here when a prototype grows beyond a single scene.

Examples:

- `ResourceSystem.ts`
- `InventorySystem.ts`
- `EventSystem.ts`
- `ShopSystem.ts`
- `CombatSystem.ts`
- `CraftingSystem.ts`
- `UpgradeSystem.ts`

Do not create complex systems before the prototype needs them.
